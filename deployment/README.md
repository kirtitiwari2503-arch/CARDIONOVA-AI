# CARDIONOVA AI - Deployment Guide

## Requirements

- Python 3.10+
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
python -m streamlit run streamlit_app.py
```
